# Basic Keylogger 🔐

A simple Python keylogger that records keystrokes locally to demonstrate how keyboard event capturing works.

## Features

* Logs keystrokes to a local file
* Real-time key capture
* Clean and readable logs
* Safe and transparent behavior

## Installation

```bash id="9u3q1l"
git clone https://github.com/yourusername/basic-keylogger.git
cd basic-keylogger
pip install -r requirements.txt
```

## Usage

```bash id="i6b1z7"
python main.py
```

Press keys and they will be logged into `logs/keylog.txt`.
